## Hvordan bidra

Du må ha følgende klart op forhånd:
* [Hugo Extended v0.144+](https://gohugo.io/installation/)
* [Node v22+](https://nodejs.org/en/download/)
* [Go v1.24+](https://go.dev/doc/install)

Deretter kjør følgende kode for å laste ned nødvendige bibliotek:
```bash
npm install
```

Deretter kan du kjøre nettsiden i developer modus:
```bash
npm run dev
```

Om du vil, så kan du også bygge nettstedet lokalt. Dette er handy for å sjekke om den vil vises korrekt på nettsiden før du comitter til repositorien.
```bash
npm run build
```
